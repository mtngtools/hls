# @mtngtools/hls-cli

CLI tools for HLS utilities.

## Overview

Command-line interface for common HLS operations.

## Installation

```bash
pnpm add -g @mtngtools/hls-cli
```

## Usage

```bash
hls transfer --source <url> --destination <path>
```

## Features

- Unified CLI interface
- Common HLS operations
- Integration with other packages
