# @mtngtools/hls-types

Core types and interfaces for HLS utilities.

## Overview

This package provides shared TypeScript types and interfaces used across the HLS utilities monorepo.

## Installation

```bash
pnpm add @mtngtools/hls-types
```

## Usage

```typescript
import type { MasterManifest, Variant, Chunk } from '@mtngtools/hls-types';
```

## Features

- Zero dependencies
- Runtime-agnostic types
- Comprehensive HLS manifest types
- Plugin system interfaces
